# STRATEGY. | Tactical Task Management

A high-performance, full-stack Todo application built with a **C# ASP.NET Core Web API** backend and a **React + Tailwind CSS v4** frontend.

## üöÄ Getting Started

### 1. Backend
1. Open the solution in **Visual Studio**.
2. Press `F5` to run the API.
3. Ensure it is running on `https://localhost:5001`.

### 2. Frontend
1. Navigate to the `my-todo-frontend` folder.
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev`
4. Open `http://localhost:5173` in your browser.
